# Frontend (React + Vite)

## Setup

1. Copy `.env.example` to `.env`.
2. Install dependencies:
   npm install
3. Start dev server:
   npm run dev

The app expects backend API at `VITE_API_BASE_URL`.

## Environment Variables

- `VITE_API_BASE_URL`: backend base URL (default in code: `http://127.0.0.1:8000/api/v1`)
